📱 操作步驟

  1. 用手電筒照亮 ESP32 的光敏電阻（GPIO34）
  2. 左右搖動手機 3 次 啟動控制器
  3. 使用方向鍵 ⬆️⬇️⬅️➡️ 控制鬼魂移動
  4. 光線不足時無法移動，記得保持手電筒照亮！
🔧 ESP32 配置

硬體連接

  • 📍 光敏電阻: GPIO 34 (ADC1)
  • 📍 WiFi: NetArt
  • 📍 固定 IP: 192.168.100.201 ~ 209

程式上傳

  1. 開啟 esp32-light-sensor.ino
  2. 修改 DEVICE_ID 和 local_IP
  3. 上傳到 ESP32
  4. 用手電筒測試光線感應
